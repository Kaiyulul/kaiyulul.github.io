## 🛠 set-up

1. Install the dependencies

   ```sh
   npm install
   ```

2. Start the development server

   ```sh
   npm start
   ```

## 🚀 build and run for production

1. Generate a full static production build, OR

   ```sh
   npm run build
   ```
   
2. Deploy straight into gh-pages branch

   ```sh
   npm run deploy
   ```
